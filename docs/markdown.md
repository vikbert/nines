## Generate TOC
use the simple `go` script to generate the `TOC` from any markdown file.

```bash
$ curl https://raw.githubusercontent.com/ekalinin/github-markdown-toc/master/gh-md-toc -o toc
$ chmod a+x toc
$ mv toc /usr/local/bin
$ toc README.md
```
